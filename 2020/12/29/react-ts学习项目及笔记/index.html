使用.env文件配置全局环境变量

.env 全局默认配置文件，不论什么环境都会加载合并

.env.development 开发环境下的配置文件

.env.production 生产环境下的配置文件

  • vue 以 **VUE_APP_ **开头
  • react 以**REACT_APP_ **开头

在**.gitignore **里面要将环境变量配置文件忽略上传
