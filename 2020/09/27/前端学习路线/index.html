六、Node(不用学太深)

包管理

  • npm
    npm install (安装单个软件包)
    npm update (更新软件包)
    npm run (运行任务)
    npm uninstall (卸载 npm 软件包)
    -g标志可以执行全局安装
    -S就是–save的简写
    -D就是–save-dev 这样安装的包的名称及版本号就会存在package.json的devDependencies这个里面，而–save会将包的名称及版本号放在dependencies里面。
